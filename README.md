# 🐕 Bagel's Customs Academy

**Where Good Dogs Help You Pass the Broker Exam!**

A whimsical customs broker exam prep tool with **295 real CBP exam questions** from official exams (2023-2025).

## 🎯 Features

- **3 Study Modes**:
  - 🔥 Speed Round (10 min, random questions)
  - 📚 Open Study (all questions, shuffled)
  - 📖 Linear Study (chronological, newest first)

- **Smart Tracking**:
  - Timer with countdown and time-per-question tracking
  - Results dashboard with detailed breakdown
  - Recurring errors tracking (localStorage persistence)
  - Session history with accuracy and duration
  - Category performance breakdown

- **Study Resources**:
  - Customs Fun Facts (strange regulations, arcane HTS codes, exam trivia)
  - Test Prep Guide (common CFR references, study tips)
  - Encouragement & Trade Puns throughout

## 📊 Question Bank

- **295 questions** from official CBP exams
- **6 exam periods**: October 2025, April 2025, October 2024, April 2024, October 2023, April 2023
- **Categories**: Entry/Entry Summary, Classification, Valuation, Broker Compliance, Practical Exercise, Modernized Drawback

## 🎓 About the CBLE

The Customs Broker License Examination (CBLE) is:
- 80 multiple-choice questions
- 4.5 hours
- Open book (HTSUS, 19 CFR, ACE documents)
- 75% passing score required
